# Intellij Keyboard Shortcut Cheat Sheet


## Window Navigation

### Open Tool Window

| Window  | Short Cut |
| -- | -- |
| Project | ⌘ + 1 |
| Bookmarks | ⌘ + 2 |
| Find | ⌘ + 3 |
| Run | ⌘ + 4 |
| Debug | ⌘ + 5 |
| Problems | ⌘ + 6 |
| Structure | ⌘ + 7 |
| Services | ⌘ + 8 |
| Git | ⌘ + 9 |
| Commit | ⌘ + 0 |

### View Recent Files

Also shows all tool windows shortcuts.

| Window  | Short Cut |
| --------  | -------- |
| Recent Files | ⌘ + E |
| Recent Locations | Shift + ⌘ + E |
